Burp Suite

Burp Suite是一款集成化的渗透测试工具，包含了很多功能，可以帮助我们高效地完成对Web应用程序的渗透测试和攻击。

Burp Suite由Java语言编写，基于Java自身的跨平台性，使这款软件学习和使用起来更方便。Burp Suite不像其他自动化测试工具，它需要手工配置一些参数，触发一些自动化流程，然后才会开始工作。

Burp Suite是一个抓包、改包的工具

Proxy模块

<1> Options模块

用于设置代理监听、请求和响应，拦截反应，匹配和替换，ssl等
  • Proxy Listeners:
  1. 默认情况下，Burp默认监听127.0.0.1 地址8080作为代理服务器,要使用这个监听器，需要配置浏览器使用127.0.0.1：8080作为代理服务器
  • Intercept Client Requests:
  1. 配置拦截规则，设置拦截的匹配规则
  2. 规则可以为：域名、IP地址、协议、HTTP方法、URL，文件拓展名，参数、cookie、头/主题内容、状态代码等
  • Intercept Server Responses:

  1. 基于服务端配置拦截规则，设置拦截的匹配规则

  2. Intercept request based on the following rules 为选中状态时，burpsuite会匹配响应包,配置列表中的规则进行拦截或转发

  • Response Modification:

  1. 用于执行响应的自动修改

  2. 可以使用这些选项来自动修改HTML应用程序响应中匹配的内容

<2>http history 模块

记录经过代理服务器的所有请求

<3>intercept 模块

显示修改http请求及应答，可以发送到其他模块
  • Forward:发送数据
  • drop:丢弃请求包
  • intercept is off/on:拦截开关
  • action:右键功能
